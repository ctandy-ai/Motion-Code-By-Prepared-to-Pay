## Step 2: Compress Videos Locally (FREE!)

### Option A: Using HandBrake (Easiest)
1. Download HandBrake: https://handbrake.fr/
2. Settings:
   - Preset: "Fast 720p30"
   - Video Codec: H.264
   - Quality: RF 28
   - Save as .mp4

### Option B: Using FFmpeg (Command Line)
```bash
ffmpeg -i input.MOV -vf "scale=-2:720" -c:v libx264 -crf 28 -b:v 1.5M -c:a aac -b:a 128k output.mp4
```
